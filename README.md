19억 투자를 유치하고 22명 규모의 팀으로 성장할 때까지 필요한 기술 역량을 책임졌던 경험이 있습니다.

**쓰레기는 본사람이 줍는다 라는 마인드**를 가지고 있습니다.
예시로 에러 알림 or 운영 이슈가 있는 경우 식별 > 이슈 제기 > 원인 분석 > 원인 해결 단계로 긴급한 이슈를 빠르게 해소합니다.

**반복되는 일을 자동화하고 비효율적인 프로세스를 개선하는 일을 좋아합니다**. 운영팀의 단순/루틴 업무 시간을 줄이기 위해 지속적으로 어드민을 개발하였습니다.
